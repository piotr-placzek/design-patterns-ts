My examples of design patterns described at [https://refactoring.guru/](https://refactoring.guru/) written in Typescript.

### How to run examples?
1. Install `nodejs` and `npm`.
1. Clone this repository.
1. Install project's dependencies with `npm i`
1. Run selected example:
    1. +`npm run factory-method`
    1. `npm run abstract-factory`
    1. +`npm run builder`
    1. +`npm run prototype`
    1. +`npm run singleton`
    1. +`npm run decorator`
    1. +`npm run flyweight`
    1. +`npm run bridge`
    1. +`npm run proxy`
    1. +`npm run chain-of-responsibility`
    1. +`npm run iterator`
    1. +`npm run observer`
    1. +`npm run command`
    1. +`npm run strategy`
    1. +`npm run visitor`
    1. `npm run adapter`
    1. `npm run composite`
    1. `npm run facade`
    1. `npm run mediator`
    1. `npm run memento`
    1. `npm run state`
    1. `npm run template-method`
1. Do not forget read the source code ;)
